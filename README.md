## 📦 Overview

**NxtTrendz** is a feature-rich e-commerce web application that mimics platforms like Flipkart/Amazon. It includes:
- Product listings
- Detailed product views
- Cart management
- Protected routes
- **🔐 Payment workflow integration**

This enhancement includes a working payment confirmation flow (mock/Stripe demo) simulating a real-world checkout experience.

- /cart – Protected

- /payment-success – Protected
